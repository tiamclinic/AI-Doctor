## 概要

医師（または院スタッフ）が、パーツ別の説明文を**ブラウザから編集・保存**できる管理 UI を提供する。自前 Next `/admin` でも、Firebase Studio / Retool / 既存 Headless CMS でも可。**認証は必須**（院メールドメイン、Google Workspace のみ等）。

## ゴール / 受け入れ基準

- [ ] T-13 で定義したスキーマに沿った**フォーム**（パーツ切替タブ or サイドナビ）
- [ ] プレビュー（読み取り専用レンダリング）で結果画面に近い見え方を確認可能
- [ ] 保存時にバリデーション（最大文字数、禁止タグ、XSS 対策）
- [ ] 監査用に**更新者・更新日時**を保存（可能なら）
- [ ] 本番 URL は検索エンジンに乗せない（`noindex`、Basic 認証、IP 制限のいずれか）
