# requirements.md §11 受け入れ基準 — 証跡マトリクス

| # | 受け入れ基準（要約） | 自動 / 手動 | 証跡・コメント |
| - | -------------------- | ----------- | -------------- |
| 1 | 写真アップロードから 60 秒以内に全結果（90% ケース） | 手動 + 計測 | [10-qa-test-cases.md](./10-qa-test-cases.md) メモ欄。複数端末で記録。 |
| 2 | 顔検出成功時に総合スコアが矛盾なく表示 | 自動寄り | Vitest（`goldenRatio` / `scoring`）+ 手動スモーク |
| 3 | 診断文に GPT 頻出禁止語が含まれない | 自動 + 手動 | `forbiddenWords` テスト + サンプル目視 |
| 4 | シェアカードが SNS に投稿できレイアウト崩れなし | 手動 | 実機投稿・プレビュー画像保存 |
| 5 | AI 理想顔が生成され本人写真と並べて表示 | 手動 | スクリーンショット |
| 6 | 利用規約・PP への同意なしに診断が始まらない | **コード** | `ConsentDialog` で規約チェック必須。アップロードは同意後のみ。 |
| 7 | iOS Safari / Android Chrome で全機能動作 | 手動 | QA 表の環境マトリクス |
| 8 | OpenAI API キーがクライアントバンドルに含まれない | 静的 | `import "server-only"` + Route Handler のみ + grep 手順（QA 表 S1） |
