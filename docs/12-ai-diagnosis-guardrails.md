## 概要

§4.9.1 に基づき、**OpenAI 診断文に具体的な施術名・医療行為の推奨が含まれない**ことを、プロンプト・禁止語・事後スキャンの多層で担保する。院方の施術説明は **T-13 以降のドクター記述**に分離する。

## ゴール / 受け入れ基準

- [ ] システムプロンプト／few-shot に「施術名・注射・切開・○○リフト等の列挙禁止」を明記
- [ ] `forbiddenWords` または専用スキャナで、代表的な**施術キーワード**を検知した場合はマスク・再生成・エラーいずれかの方針を実装（方針はチケット内で決定）
- [ ] 診断 JSON スキーマの説明文に「施術推奨は含めない」旨をコメントまたは `description` で残す
- [ ] Vitest で、施術っぽいダミー出力がフィルタされるケースを 1 件以上追加
- [ ] ドキュメント（`docs/api/` または本チケット）に「AI と院方文の分担」を明記
