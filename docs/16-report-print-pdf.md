## 概要

院内説明・持ち帰り用に、**ブラウザ印刷**で読みやすいレイアウトを提供する。理想要件として **PDF ダウンロード**（`@react-pdf` / Playwright PDF / サーバー生成等）の採用を検討する。

## ゴール / 受け入れ基準

- [ ] `@media print` で、ナビ・フッター・不要ボタンを非表示にし、**A4 縦**想定の余白・改ページを調整
- [ ] 「印刷」ボタンから `window.print()` を呼び出し可能
- [ ] 印刷プレビューで写真・スコア・AI 文・院方文が欠けない（色は印刷向けに調整可）
- [ ] （任意）PDF 1 ファイル生成＋ダウンロード。個人情報（写真）を含むため**同意文**の有無を検討
