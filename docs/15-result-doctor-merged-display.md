## 概要

同一結果画面上で、**数値スコア＋AI 診断レポート＋ドクター記述**を表示する。§4.9.4 に従い、**AI 参考**と**当院医師による説明**を色・枠・見出し・脚注で**誤解なく分離**する。

## ゴール / 受け入れ基準

- [ ] パーツごとに「AI 由来の短文（任意）」と「院方テキスト」を並べるか、院方のみ表示するかを UI で明確化（モックに合わせる）
- [ ] 全ブロック共通の**法的脚注**（医療診断ではない／院方文の責任所在）を固定表示
- [ ] 院方 API 取得失敗時も AI 結果のみで破綻しない
- [ ] ダークモード不要前提で TIAM トークンに準拠
